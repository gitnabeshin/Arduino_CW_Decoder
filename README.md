## OverView
* Original CW Decoder using Arduino Nano.
  * 2 input Mode (English/Japanese)
  * Sampling rate adjusting Mode
  * More Option Mode appendable
* Parts List
  * Telegraph key (switching device used to send Morse code)
  * Arduino Nano
  * LCD1602 (I2S)
  * Speaker
  * 4 Buttons
  * Battery
  * Card board case
